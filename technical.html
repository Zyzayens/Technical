intro

Le HTLM est le language qui permet de construire des pages web

page

La page html se definie par la presence de !doctype html et les balises HTML

<!DOCTYPE html> <html> </html>

head

La balise head sert principalement à défnir :

  • Les liens de référence pour la page
<head> </head>

Les balises <head> peuvent contenir les meta elements, le titre de la page, ainsi que les divers liens CCS pour les font, ou la feuille de style de la page.

body

La balise body sert principalement à défnir :

  • Le corps de la page et les principales infos qui seront à l'écran
<body></body>

C'est lui qui vas contenir les divers div et sections du document, tout ce qui se trouve entre ses balises est ce qui sera affiché sur l'écran.

div

La balise div sert principalement à défnir :

  • Des zones spéciale pour la mise en page
  • Des conteneur pour divers sections (images, vidéo, formulaires, etc...)
<div></div>

Les div servent en général de conteneur pour les elements et de ce fait est utilisé dans les feuilles CSS avec une classe ou une id. Sa fonction est très similaire aux balises sections.

footer

La balise footer sert principalement à défnir :

  • Le bas de page les infos de contact et les liens externes pour les réferences, et copyright.
<footer></footer>

Les balises footer font partie du corps de la page et donc est affiché par le moniteur, elle peuvent contenir toutes sortes de balises a l'interieur d'elles.
